Новостная лента с календарем «DnP NEWS SQL 3.2 + Kalendar»
 
Теперь это не Новостная лента с календарем, а
Календарь новостей и событий.
Встраивается в главную страницу сайта.

Автор: Вишневский Игорь - igor_vish@mail.ru
Требования: PHP 4-5 + MySQL
-----------------------------------------------
Порядок установки:

1) Распакуйте архив со скриптом на свой сайт.
2) Измените атрибуты папки admin/news на 777 " chmod 777 news " (если вы используете Unix систему.)
3) Создайте базу MySql если баз еще нет.
4) Сделайте SQL запрос из файла папка скрипта/admin/news.sql в созданной базе или уже имеющейся.
5) Сконфигурируйте файл настройки 'папка скрипта/admin/config.inf'
6) Зайдите через браузер в администраторский раздел 'http://ваш_сайт/папка скрипта/admin/' Имя: user пароль: user

На главной странице сайта:
В месте отображения новостей
установить код <? include("папка скрипта/news.php"); ?>
В месте отображения календаря
установить код <? include("папка скрипта/calend.php"); ?>   


Для проверки откройте через браузер http://ваш_сайт/папка скрипта/index.php
-----------------------------------------------
Возможности:

Добавление новостей:
	Добавление на прошлую и будущую дату
	Актуальность новости (Первая N-дней, автоматически отключается при оканчаниии срока)
	

Управление пользователями:
	Создание пользователей, удаление, три статуса(доступа) пользователя.
                Статус администратора позволяет редактировать html тэги.
Календарь:
	Включение-отключение приглашения "Добавить новость".
